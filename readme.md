## 紹介

  - なぜじゃんけんはつまらなくなるのか？なぜ大人数だと決着がつかないのか？
  
  - じゃんけんのルールを見つめ、あいこになる確率を下げるべく "キュン" という新しい概念を投入。

  - "キュン" はフィールドに１人だけなら必ず "勝つ" 。

  - "キュン" はフィールドに２人以上なら "負ける" 。ただし残りのメンバーが "あいこ" であれば "あいこ" となる。
